#### 分析
主体框架为Angular11，包管理工具为Yarn，如果要开发前端，需要了解一下技术：
- Angular
  - router
  - module
  - component
  - service
  - pipe
  - zone
- Angular Material（基础组件库）
- NGRX（响应式状态管理）
- RxJS（异步编程）
