#### 分析
该工程没有`pom.xml`，通过结构分析，3层结构如下：

```
.
├── java
│   ├── assembly
│   │   └── windows.xml
│   ├── build.gradle
│   ├── filters
│   │   ├── unix.properties
│   │   └── windows.properties
│   └── scripts
│       ├── control
│       ├── install
│       └── windows
└── js
    ├── assembly
    │   └── windows.xml
    ├── build.gradle
    ├── filters
    │   ├── unix.properties
    │   └── windows.properties
    └── scripts
        ├── control
        ├── init
        └── windows
```

主要是一些脚本、配置文件，猜测用于编译多种环境（`unix`、`windows`）的安装包
